SETTING DATA AWAL

  1. Isikan data perusahaan anda pada kolom-kolom yang telah tersedia

    1. Nama usaha anda
    2. Pemilik perusahaan
    3. Alamat usaha anda
    4. Nomor telepon
    5. Alamat e-mail
  2. Jika sudah terisi maka tekan tombol simpan dan lanjutkan
  3. Tekan "Ya" jika anda ingin memasukan neraca awal, tekan "Tidak" jika anda tidak ingin memasukan neraca awal.

Jika anda ingin memasukan neraca awal maka akan muncul tampilan sebagai berikut:

  1. Tombol restart jika terdapat kesalahan dalam memasukan data neraca awal
  2. Tombol untuk memilih tanggal
  3. Tombol untuk memilih akun debet
  4. Kolom untuk memasukan nominal debet
  5. Tombol untuk memilih akun kredit
  6. Kolom untuk memasukan nominal kredit
  7. Tombol untuk menambah akun debet dan akun kredit
  8. Tombol simpan, untuk meniympan data neraca awal

  6. Menu tentang, berisi informasi tentang aplikasi

MEMASUKKAN DATA JURNAL

  1. Tombol jika anda ingin menambah jurnal
  2. Tombol restart, untuk mengulang jika terjadi kesalahan
  3. Tombol tanggal, pilih tanggal terjadinya transaksi
  4. Kolom keterangan, isikan keterangan transaksi pada kolom tersebut
  5. Pilih jenis transaksi, yang akan anda masukkan.
  6. Pilih akun debet yang sesuai dengan transaksi yang anda masukan
  7. Masukan nominal debet
  8. Pilih akun kredit yang sesuai dengan transaksi yang anda masukan
  9. Masukan nominal kredit
  10. Tombol tambah akun debet dan tambah akun kredit digunakan jika transaksi melibatkan beberapa akun.
  11. Tombol simpan untuk menyimpan transaksi.

    12. NOTE: Jika nominal yang anda masukan tidak seimbang akan muncul pemberitahuan

SUB MENU LAPORAN

  1. Tombol untuk melihat rekapitulasi jurnal selama periode berjalan.
  2. Tombol untuk melihat Neraca saldo
  3. Tombol untuk melihat lapora Laba/rugi
  4. Tombol untuk melihat laporan perubahan ekuitas
  5. Tombol untuk melihat Neraca
  6. Tombol untuk melihat Laporan Arus Kas
  7. Tombol untuk covert PDF, mengubah file menjadi PDF.

SUB MENU PENGATURAN

  1. Tombol untuk mengatur atau mengedit identitas perusahaan
  2. Tombol untuk melihat atau menambah akun
  3. Tombol untuk mencadangkan seluruh data yang telah dimasukan.
  4. Tombol untuk mengembalikan data seperti semula setelah dihapus.
  5. Tombol untuk menghapus semua data yang telah dimasukan.

PENGATURAN PERUSAHAAN

  1. Tekan tombol "Edit" untuk mengedit identitas perusahaan.
  2. Tekan tombol "Simpan" untuk menyimpan perubahan identias perusahaan

PENGATURAN AKUN

  1. Pilih jenis akun mana yang akan anda tambahkan
  2. Tombol untuk menambah akun
  3. Masukkan kode akun dan nama akun yang akan anda tambahkan di kolom yang telah tersedia
  4. Tekan tombol "Tambahkan" untuk menyimpan akun yang anda tambahkan.

PENGATURAN BACKUP DATA

  1. Tekan tombol "Backup Data"
  2. Tekan tombol "Ya" untuk backup/mencadangakan semua data, jika anda tidak ingin mencadangkan maka tekan "Tidak"

PENGATURAN RESTORE DATA

  1. Tekan tombol "Restore" jika anda ingin mengebalikan data yang telah anda hapus
  2. Tekan tombol "Ya" jika anda yakin ingin merestore data, tekan tombol "Tidak" untuk membatalkan

PENGATURAN HAPUS SEMUA DATA

  1. Tekan tombol "Hapus semua data" jika anda ingin menghapus semua data.
  2. Tekan tombol "Ya" jika anda yakin ingin menghapus semua data, tekan tombol "Tidak" untuk membatalkan.

TAMPILAN MENU MATERI

  1. Daftar materi yang bisa anda pilih
  2. Tombol yang dapat anda pilih untuk mengetahui manfaat dari masing-masing laporan keuangan
  3. Tombol yang dapat anda pilih untuk mengetahui penjelasan dari masing-masing jenis-jenis transaki
  4. Tombol yang dapat anda pilih untuk mengetahui penjelasan dari masing-masing jenis transaski penyesuaian.
